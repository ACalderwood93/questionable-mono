# Questionable Mono

A TypeScript monorepo powered by Turborepo, Biome, and TypeScript.

## Getting Started

### Install dependencies

```bash
pnpm install
```

### Build

```bash
pnpm build
```

### Develop

```bash
pnpm dev
```

### Lint

```bash
pnpm lint
```

### Format

```bash
pnpm format
```

### Type Check

```bash
pnpm type-check
```

## Tech Stack

- **Turborepo** - High-performance build system for JavaScript and TypeScript monorepos
- **Biome** - Fast formatter and linter for JavaScript and TypeScript
- **TypeScript** - Typed JavaScript at any scale
- **pnpm** - Fast, disk space efficient package manager
